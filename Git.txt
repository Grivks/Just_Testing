REPOSITORY
Repository is a directory or storage space where your projects can live that contain folders and files, images, videos, spreadsheets, data sets, etc.
How to create repository:
- At the homepage, in the upper right corner click + and then select New repository.
- Name the repository.
- Write description of the repository (optional).
- Select Add a README File to initialize the repository.
- Then click Create Repository button.

Branch
When creating a repository on GitHub with content, the repository is initialized with a single branch, called the default branch named Master. Branching is the way to work on different versions of a repository at one time.
How to create Branch:
- Go to the repository.
- Click the dropdown at the top of the file list that says branch: master
- Write the branch name in the text box.
- Click Create branch.
